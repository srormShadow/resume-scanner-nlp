# Resume Scanner using Keyword Extraction

This project simulates a basic Applicant Tracking System (ATS) by extracting keywords from resumes and job descriptions using NLP techniques and calculating a match score.

## Features
- Resume text extraction (PDF/DOCX)
- Text preprocessing
- Keyword extraction using TF-IDF
- Resume vs Job Description matching

## Tech Stack
- Python
- NLTK
- scikit-learn
